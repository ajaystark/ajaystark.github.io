# Video Assets

## freedom-fest-video.mp4

This file is referenced in the HTML but needs to be added.

**Recommended video specifications:**
- Format: MP4 (H.264)
- Resolution: 1920x1080 or higher
- Duration: 15-30 seconds loop
- Content: Freedom Fest activities (paragliding, people enjoying nature, community activities)
- Style: Dynamic, engaging footage that captures the adventure spirit

**To add the video:**
1. Place your video file in this directory as `freedom-fest-video.mp4`
2. The video will automatically play in the background of the hero section
3. Video is set to autoplay, muted, and loop for optimal user experience

If you don't have a video ready, the hero section will still display properly with a gradient background.
